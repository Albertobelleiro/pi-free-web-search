## Questions and Help

If you need help using the project, open a GitHub Discussion (if enabled) or open an issue using the question/help template.

## Bug Reports

Use the bug issue template and include:

- environment (OS, browser family, Pi version)
- reproduction steps
- expected vs actual behavior
- logs/output snippets

## Security Issues

For security vulnerabilities, follow [SECURITY.md](./SECURITY.md) and report privately.
